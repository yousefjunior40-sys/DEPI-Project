# 🚚 Truck Spot

Truck Spot is a modern web application that helps users discover nearby food trucks, browse menus, place orders, and leave reviews through a clean and responsive interface.

## ✨ Features

- 🔍 Browse food trucks
- 📍 View truck locations
- 🍔 Explore menus
- 🛒 Place and manage orders
- ⭐ Rate and review food trucks
- 👤 User authentication
- 📱 Fully responsive design

## 🛠️ Built With

- React
- javaScript
- Vite
- Tailwind CSS
- React Router
- Local Storage
